Aula extra de Git e Github para Fixação

Comandos do Git

git init - para iniciar um repositório

git add "file" ou git add . ou git add * ou git add -A - para adicionar novos arquivos ou novas modificações para o git vigiar

git commit -m "mensagem" - para criar um novo commit

git status - para verificar arquivos que ainda não foram commitados

git log - para listar todos os commits de uma branch

git config user.name "nome de usuario" - para configurar nome de usuário

git config user.email - para configurar email
